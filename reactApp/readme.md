## 内容介绍
*  这是一个介绍如何使用react的小示例，本例中结合时下流行的redux库，演示
   在react中怎么利用redux实现单向数据流。
*  更改jsx代码会自动刷新页面
*  本示例中省略了action中数据获取的过程（比如ajax),重点突出action=>dispatch=>store=>view 这个过程。

* 开发调试
```javascript 
   npm install && npm start
```

### 打包发布

```javascript 
   npm run build
```
